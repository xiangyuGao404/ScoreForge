# ScoreForge

> 面向家长的 AI 学情诊断工具——拍一张试卷，AI 告诉你孩子最该补哪里、怎么补、补到什么程度算过关。

## 项目简介

ScoreForge 是一款面向家长的 AI 驱动学情诊断与精准练习系统。家长上传孩子的阶段性测试试卷，AI 自动识别并分析薄弱环节，按"提分性价比"排序推荐练习方向，生成针对性题目（在线作答或 PDF 打印），并持续追踪掌握进度。

## 核心功能

- **试卷诊断**：拍照上传 → AI 预识别 → 家长确认 → 薄弱点分析
- **智能出题**：按薄弱方向生成针对性练习题（在线/PDF 双模式）
- **掌握追踪**：AI 综合判断掌握程度，主动建议"已掌握"或"继续练"
- **虚拟教师团**：数学老师、道法老师、历史老师、班主任，基于学生数据个性化答疑
- **多孩子管理**：一个账号支持多个孩子，独立追踪各自学情

## 支持科目

- 数学（主要）
- 道德与法治
- 历史
- 语文、英语（V2.0 规划）
