## P0 — 必须先修（阻塞核心流程）

### 任务 1：修复 `getWeaknesses` 响应字段映射

**文件**：`src/utils/service.ts` 第 190-202 行
**问题**：后端返回 `{data: {weaknesses: [{weakness_id, knowledge_point, star_rating, reason, status, mastery_score, practice_days, recommended_days}]}}`。前端 `Weakness` 接口需要 `student_id` 字段，但后端不返回该字段。
**修复**：
```ts
// service.ts 的 getWeaknesses 函数
// 后端返回的是 {weaknesses: [...]}，每项不含 student_id
// 需要把当前 student_id 注入进去
export async function getWeaknesses(studentId?: string) {
  if (USE_MOCK) {
    await mock.delay(300)
    return { code: 0, data: mock.mockWeaknessList }
  }
  const sid = studentId || uni.getStorageSync('currentStudentId') || ''
  const res = await request<{ weaknesses: any[] }>({
    url: `/weaknesses?student_id=${sid}`,
  })
  // 注入 student_id（后端不返回该字段）
  const list = (res.data.weaknesses || []).map(w => ({
    ...w,
    student_id: sid,
  }))
  return { code: res.code, data: list }
}
```

## 后端接口速查（前端对接用）

### 认证

| 接口 | 方法 | 请求 | 响应 |
|------|------|------|------|
| `/api/v1/auth/send-code` | POST | `{phone}` | `{code:0}` |
| `/api/v1/auth/login` | POST | `{phone, code}` | `{data:{token, user_id, nickname, user_level}}` |

### 学生

| 接口 | 方法 | 请求 | 响应 |
|------|------|------|------|
| `/api/v1/students` | GET | - | `{data:{students:[{id, name, grade, school, subjects}]}}` |
| `/api/v1/students` | POST | `{name, grade, subjects}` | `{data:{id, name, ...}}` |
| `/api/v1/students/{id}` | PUT | `{name?, grade?, school?, subjects?}` | `{data:{...}}` |

**注意**：`grade` 只接受 `"初一"` / `"初二"` / `"初三"`

### 试卷

| 接口 | 方法 | 请求 | 响应 |
|------|------|------|------|
| `/api/v1/exams/upload` | POST | multipart/form-data（student_id, subject, exam_name, total_score, actual_score, images） | `{data:{exam_id, status}}` |
| `/api/v1/exams/{id}/recognition` | GET | - | `{data:{exam_id, status, questions:[{question_no, question_content, is_correct, score_got, score_total, confidence}]}}` |
| `/api/v1/exams/{id}/confirm` | POST | `{questions:[{question_no, is_correct, score_got}]}` | `{data:{exam_id, status}}` |
| `/api/v1/exams/{id}/analysis` | GET | - | `{data:{exam_id, subject, total_score, actual_score, weaknesses:[{weakness_id, knowledge_point, star_rating, reason, status}]}}` |

### 薄弱点

| 接口 | 方法 | 请求 | 响应 |
|------|------|------|------|
| `/api/v1/weaknesses?student_id=xxx` | GET | query: student_id | `{data:{weaknesses:[{weakness_id, knowledge_point, star_rating, reason, status, mastery_score, practice_days, recommended_days}]}}` |
| `/api/v1/weaknesses/{id}/master` | POST | - | `{message:"已标记为掌握"}` |

**⚠️ 注意**：`getWeaknesses` 返回的是 `{weaknesses: [...]}`，不是 `[...]`。前端需要 `res.data.weaknesses`。

### 练习

| 接口 | 方法 | 请求 | 响应 |
|------|------|------|------|
| `/api/v1/practice/generate` | POST | `{student_id, weakness_id, mode, question_count}` | `{data:{session_id, mode, status}}` |
| `/api/v1/practice/{id}/questions` | GET | - | `{data:{session_id, weakness, questions:[{question_no, question_content, reference_answer, solution_detail, difficulty}]}}` |
| `/api/v1/practice/{id}/submit` | POST | `{results:[{question_no, student_answer, is_correct}]}` | `{data:{session_id, correct_rate, assessment_status}}` |
| `/api/v1/practice/{id}/assessment` | GET | - | `{data:{mastery_score, trend, error_pattern, recommendation, suggested_days, suggestion_detail, history}}` |
