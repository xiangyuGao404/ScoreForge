## 一、试卷上传与识别（异步）

### 流程

```
上传图片 → POST /upload-by-urls → 轮询 GET /recognition → 确认 → 轮询 GET /analysis
```

### ① 单图上传

```
POST /api/v1/exams/upload-image
Content-Type: multipart/form-data
Authorization: Bearer {token}
Body: image (单个文件)
```

**Response**：
```json
{"code": 0, "data": {"url": "./uploads/xxx.jpg", "filename": "xxx.jpg"}}
```

### ② 提交试卷（异步，立即返回）

```
POST /api/v1/exams/upload-by-urls
Content-Type: application/json
Authorization: Bearer {token}
```

**Request**：
```json
{
  "student_id": "uuid",
  "subject": "math",
  "exam_name": "月考",
  "total_score": 120,
  "actual_score": 73,
  "image_urls": ["./uploads/xxx.jpg", "./uploads/yyy.jpg"]
}
```

**Response**（立即返回，不等 OCR）：
```json
{"code": 0, "data": {"exam_id": "uuid", "status": "recognizing", "message": "试卷已接收，正在识别中"}}
```

### ③ 轮询识别结果

```
GET /api/v1/exams/{exam_id}/recognition
Authorization: Bearer {token}
```

状态为 `recognized` 时，`questions` 有数据：

```json
{
  "code": 0,
  "data": {
    "exam_id": "uuid",
    "status": "recognized",
    "questions": [
      {"question_no": 1, "question_content": "题目内容", "is_correct": true, "score_got": 5, "score_total": 5, "confidence": 0.95, "parent_verified": false}
    ]
  }
}
```

### ④ 确认识别（异步，立即返回）

```
POST /api/v1/exams/{exam_id}/confirm
Content-Type: application/json
Authorization: Bearer {token}
```

**Request**：
```json
{
  "questions": [
    {"question_no": 1, "is_correct": true, "score_got": 5, "parent_verified": true},
    {"question_no": 2, "is_correct": false, "score_got": 0, "student_answer": "x=3", "parent_verified": true}
  ]
}
```

**Response**（立即返回，不等分析）：
```json
{"code": 0, "data": {"exam_id": "uuid", "status": "analyzing"}, "message": "已确认，AI 正在分析薄弱点"}
```

### ⑤ 轮询分析结果

```
GET /api/v1/exams/{exam_id}/analysis
Authorization: Bearer {token}
```

**分析中**：
```json
{"code": 0, "data": {"exam_id": "uuid", "subject": "math", "total_score": 120, "actual_score": 73, "weaknesses": []}}
```

**分析完成**（`weaknesses` 有数据）：
```json
{
  "code": 0,
  "data": {
    "exam_id": "uuid",
    "subject": "math",
    "total_score": 120,
    "actual_score": 73,
    "weaknesses": [
      {"weakness_id": "uuid", "knowledge_point": "函数基础概念", "star_rating": 5, "reason": "说明", "status": "not_started"}
    ]
  }
}
```

## 二、练习出题与评估（异步）

### 流程

```
POST /generate → 轮询 GET /questions → POST /submit → 轮询 GET /assessment
```

### ① 生成题目（异步，立即返回）

```
POST /api/v1/practice/generate
Content-Type: application/json
Authorization: Bearer {token}
```

**Request**：
```json
{
  "student_id": "uuid",
  "weakness_id": "uuid",
  "mode": "online",
  "question_count": 5
}
```

**Response**（立即返回，不等 AI 出题）：
```json
{"code": 0, "data": {"session_id": "uuid", "mode": "online", "status": "generating", "message": "正在生成题目"}}
```

### ② 轮询题目

```
GET /api/v1/practice/{session_id}/questions
Authorization: Bearer {token}
```

**生成中**：
```json
{"code": 0, "data": {"session_id": "uuid", "weakness": "函数基础概念", "questions": []}}
```

**就绪**（`questions` 有数据）：
```json
{
  "code": 0,
  "data": {
    "session_id": "uuid",
    "weakness": "函数基础概念",
    "questions": [
      {
        "question_no": 1,
        "difficulty": "basic",
        "question_content": "题目内容",
        "question_type": "choice",
        "reference_answer": "A",
        "solution_detail": "解析"
      }
    ]
  }
}
```

### ③ 提交答案（异步，立即返回）

```
POST /api/v1/practice/{session_id}/submit
Content-Type: application/json
Authorization: Bearer {token}
```

**Request**：
```json
{
  "results": [
    {"question_no": 1, "student_answer": "A", "is_correct": true},
    {"question_no": 2, "student_answer": "x=3", "is_correct": false}
  ]
}
```

**Response**（立即返回，不等评估）：
```json
{"code": 0, "data": {"session_id": "uuid", "correct_rate": 0.6, "assessment_status": "assessing", "message": "正在评估掌握程度"}}
```

### ④ 轮询评估结果

```
GET /api/v1/practice/{session_id}/assessment
Authorization: Bearer {token}
```

**完成**：
```json
{
  "code": 0,
  "data": {
    "session_id": "uuid",
    "mastery_score": 75,
    "trend": "rising",
    "error_pattern": "概念不清",
    "recommendation": "continue",
    "suggested_days": 2,
    "suggestion_detail": "建议继续练习，巩固基础知识",
    "history": [
      {"date": "2026-06-28", "correct_rate": 0.6, "mastery_score": 75}
    ]
  }
}
```

## 三、轮询工具

```typescript
export function poll<T>(
  fn: () => Promise<T>,
  check: (data: T) => boolean,
  interval = 2500,
  maxAttempts = 60
): Promise<T> {
  return new Promise((resolve, reject) => {
    let attempts = 0
    const timer = setInterval(async () => {
      attempts++
      try {
        const data = await fn()
        if (check(data)) {
          clearInterval(timer)
          resolve(data)
        } else if (attempts >= maxAttempts) {
          clearInterval(timer)
          reject(new Error('处理超时，请稍后重试'))
        }
      } catch (e) {
        clearInterval(timer)
        reject(e)
      }
    }, interval)
  })
}
```

### 使用示例

```typescript
// 轮询识别结果
await poll(
  () => getRecognition(examId),
  (data) => data.data.status === 'recognized'
)

// 轮询分析结果
await poll(
  () => getAnalysis(examId),
  (data) => data.data.status === 'analyzed'
)

// 轮询题目
await poll(
  () => getPracticeQuestions(sessionId),
  (data) => data.data.status === 'ready'
)

// 轮询评估
await poll(
  () => getAssessment(sessionId),
  (data) => data.data.status === 'assessed'
)
```

## 四、状态说明

```
试卷: RECOGNIZING → RECOGNIZED → ANALYZING → ANALYZED
练习: GENERATING → READY → ASSESSING → ASSESSED
```
