## P0 — 必须修（阻塞前端核心流程）

### 任务 1：`GET /weaknesses` 响应补充 `student_id` 字段

**文件**：`backend/app/api/v1/weaknesses.py` 第 52-68 行
**问题**：前端 `weakness/index.vue` 的 `startPractice` 需要 `w.student_id` 来调用 `generatePractice`，但后端响应中没有该字段。
**修复**：在返回的 dict 中添加 `"student_id": str(w.student_id)`：
```python
return APIResponse(
    data={
        "weaknesses": [
            {
                "weakness_id": str(w.id),
                "student_id": str(w.student_id),  # ← 新增
                "knowledge_point": kp.name,
                "star_rating": w.star_rating,
                "reason": w.reason or "",
                "status": w.status.value,
                "mastery_score": w.mastery_score,
                "practice_days": w.practice_days,
                "recommended_days": w.recommended_days,
            }
            for w, kp in weaknesses
        ]
    }
)
```

### 任务 2：实现 `GET /students/{id}` 单个学生查询

**文件**：`backend/app/api/v1/students.py`
**问题**：前端需要获取单个学生详情（如编辑页面），当前只有列表接口。
**新增端点**：
```python
@router.get("/{student_id}", response_model=APIResponse[StudentResponse])
async def get_student(
    student_id: uuid.UUID,
    current_user: User = Depends(get_current_user),
    db: AsyncSession = Depends(get_db),
):
    result = await db.execute(select(Student).where(Student.id == student_id))
    student = result.scalar_one_or_none()
    if student is None:
        raise NotFoundException("孩子档案不存在")
    if student.user_id != current_user.id:
        raise ForbiddenException("无权查看该孩子档案")
    return APIResponse(data=StudentResponse.model_validate(student))
```

---

### 任务 3：实现 `DELETE /students/{id}` 删除学生

**文件**：`backend/app/api/v1/students.py`
**问题**：前端设置页需要删除孩子功能。
**新增端点**：
```python
@router.delete("/{student_id}", response_model=APIResponse)
async def delete_student(
    student_id: uuid.UUID,
    current_user: User = Depends(get_current_user),
    db: AsyncSession = Depends(get_db),
):
    result = await db.execute(select(Student).where(Student.id == student_id))
    student = result.scalar_one_or_none()
    if student is None:
        raise NotFoundException("孩子档案不存在")
    if student.user_id != current_user.id:
        raise ForbiddenException("无权删除该孩子档案")
    await db.delete(student)
    await db.flush()
    return APIResponse(message="已删除")
```

### 任务 4：实现教师对话端点 `POST /chat/send` + `GET /chat/history`

**文件**：新建 `backend/app/api/v1/chat.py`，注册到 `__init__.py`
**问题**：`ChatMessage` 模型已定义，但无路由、无 Schema、无服务。
**需要新建**：
- `app/schemas/chat.py` — ChatSendRequest、ChatHistoryResponse
- `app/api/v1/chat.py` — 两个端点
- `app/services/chat_service.py` — 调用 AI 服务 + 角色 Prompt 注入

**接口定义**：

```
POST /api/v1/chat/send
Body: {student_id, teacher_role, content}
Response: {data: {id, role:"assistant", content, created_at}}

GET /api/v1/chat/history?student_id=xxx&teacher_role=math
Response: {data: [{id, role, content, created_at}]}
```

**注意**：教师角色只接受 `math` / `politics` / `history` / `homeroom`。
